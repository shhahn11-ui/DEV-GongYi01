3단계 설치

다운로드한 매크로 키보드 빌드를 보드에 올리고 테스트하려면 다음 단계를 따라주세요.

1

다운로드

위 다운로드 버튼을 눌러 ZIP을 저장한 뒤, 압축을 해제합니다.

2

플래싱 준비

Arduino IDE에서 보드를 선택하고, 라이브러리를 맞춘 뒤 스케치를 열어 확인합니다. 필요하면 체크섬으로 파일 무결성을 검증하세요.

3

업로드 및 테스트

보드에 업로드한 뒤, 매크로 키 동작과 키맵이 원하는 대로 작동하는지 테스트합니다.

사용법

Macro-KeyBoard를 다운로드 하고 dist 폴더를 열어 exe 실행파일을 실행하세요.

  • 시리얼 연결: 보드를 USB로 연결하면 Python 유틸이 자동으로 포트를 탐색합니다.
  • 키맵 수정: 프로그램에서 KEY-1~4 입력값을 원하는 단축키 또는 문자열로 변경 후 저장하세요.
  • 입력 전송: 보드에서 스위치를 누르면 시리얼로 KEY-1~4 신호를 보내고, Python 유틸이 설정된 단축키를 실행합니다.
  • 붙여넣기 모드: 단축키 구분자가 없으면 문자열로 판단해 클립보드에 복사 후 자동 붙여넣기를 수행합니다.
  • 재연결: 연결이 끊기면 상태 표시줄에 안내가 뜨며, 케이블을 다시 연결하면 자동으로 재시도합니다.

코드 스니펫

직접 작성한 파이썬 스크립트나 아두이노 스케치를 아래에 붙여 넣어 공유하세요.

Arduino 스케치 (.ino)
void setup() {
  Serial.begin(9600);

  // D3, D5, D7 핀을 가공 GND로 설정
  pinMode(3, OUTPUT); digitalWrite(3, LOW); 
  pinMode(5, OUTPUT); digitalWrite(5, LOW); 
  pinMode(7, OUTPUT); digitalWrite(7, LOW); 

  // 버튼 설정 (D2, D4, D6, D8)
  pinMode(2, INPUT_PULLUP); // KEY-4 신호 전송
  pinMode(4, INPUT_PULLUP); // KEY-3 신호 전송
  pinMode(6, INPUT_PULLUP); // KEY-2 신호 전송
  pinMode(8, INPUT_PULLUP); // KEY-1 신호 전송
}

void loop() {
  // KEY-4 (D2)
  if (digitalRead(2) == LOW) {
    delay(20); // 첫 번째 떨림 방지
    if (digitalRead(2) == LOW) {
      Serial.println("KEY-4");
      while(digitalRead(2) == LOW); // 뗄 때까지 대기
      delay(20); // 뗄 때 발생하는 떨림 방지
    }
  }

  // KEY-3 (D4)
  if (digitalRead(4) == LOW) {
    delay(20);
    if (digitalRead(4) == LOW) {
      Serial.println("KEY-3");
      while(digitalRead(4) == LOW);
      delay(20);
    }
  }

  // KEY-2 (D6)
  if (digitalRead(6) == LOW) {
    delay(20);
    if (digitalRead(6) == LOW) {
      Serial.println("KEY-2");
      while(digitalRead(6) == LOW);
      delay(20);
    }
  }

  // KEY-1 (D8)
  if (digitalRead(8) == LOW) {
    delay(20);
    if (digitalRead(8) == LOW) {
      Serial.println("KEY-1");
      while(digitalRead(8) == LOW);
      delay(20);
    }
  }
}
Python 유틸
import tkinter as tk
from tkinter import messagebox
import serial
import serial.tools.list_ports
import pyautogui
import pyperclip 
import threading
import json
import os
import time

CONFIG_FILE = "macro_settings.json"

class MacroApp:
    def __init__(self, root):
        self.root = root
        self.root.title("매크로 패드 설정 프로그램")
        self.root.geometry("400x550")
        self.root.configure(bg="#1e1e1e")

        # 스타일 설정
        self.accent_color = "#0078d4"
        self.bg_color = "#1e1e1e"
        self.card_color = "#2d2d2d"

        # 기본 설정값 유지 (한시형 님 요청 사항)
        self.default_settings = {
            "KEY-1": "ctrl,a", # 전체선택
            "KEY-2": "ctrl,c", # 복사
            "KEY-3": "ctrl,v", # 붙여넣기
            "KEY-4": "ctrl,z"  # 되돌리기
        }
        self.settings = self.load_settings()

        # 헤더
        header = tk.Frame(root, bg=self.accent_color, height=80)
        header.pack(fill="x")
        tk.Label(header, text="나노 매크로 패드", font=("Malgun Gothic", 16, "bold"), bg=self.accent_color, fg="white").pack(pady=20)

        # 입력 영역
        self.main_frame = tk.Frame(root, bg=self.bg_color)
        self.main_frame.pack(pady=20, padx=30, fill="both", expand=True)

        self.entries = {}
        for key in ["KEY-1", "KEY-2", "KEY-3", "KEY-4"]:
            row = tk.Frame(self.main_frame, bg=self.bg_color)
            row.pack(fill="x", pady=10)
            
            tk.Label(row, text=key, font=("Malgun Gothic", 10, "bold"), bg=self.bg_color, fg=self.accent_color, width=8, anchor="w").pack(side="left")
            
            entry = tk.Entry(row, bg=self.card_color, fg="white", insertbackground="white", 
                             relief="flat", font=("Consolas", 11), borderwidth=8)
            # 설정 파일이 있으면 가져오고, 없으면 기본값 사용
            entry.insert(0, self.settings.get(key, self.default_settings[key]))
            entry.pack(side="right", fill="x", expand=True)
            self.entries[key] = entry

        # 저장 버튼
        self.save_btn = tk.Button(root, text="설정 저장하기", command=self.save_settings, 
                                  bg=self.accent_color, fg="white", font=("Malgun Gothic", 10, "bold"),
                                  relief="flat", padx=20, pady=10, cursor="hand2")
        self.save_btn.pack(pady=20)

        # 상태 바
        self.status_frame = tk.Frame(root, bg="#2d2d2d", height=30)
        self.status_frame.pack(fill="x", side="bottom")
        self.status_label = tk.Label(self.status_frame, text="장치를 찾는 중...", 
                                     font=("Malgun Gothic", 9), bg="#2d2d2d", fg="#aaaaaa")
        self.status_label.pack(pady=5)

        self.running = True
        self.thread = threading.Thread(target=self.serial_loop, daemon=True)
        self.thread.start()

    def load_settings(self):
        if os.path.exists(CONFIG_FILE):
            try:
                with open(CONFIG_FILE, "r", encoding="utf-8") as f:
                    return json.load(f)
            except: return self.default_settings
        return self.default_settings

    def save_settings(self):
        for key in self.entries:
            self.settings[key] = self.entries[key].get().strip()
        with open(CONFIG_FILE, "w", encoding="utf-8") as f:
            json.dump(self.settings, f)
        messagebox.showinfo("저장 완료", "설정이 저장되었습니다!")

    def serial_loop(self):
        while self.running:
            target_port = None
            for p in serial.tools.list_ports.comports():
                if any(x in p.description.upper() for x in ['USB', 'CH340', 'ARDUINO', 'SERIAL']):
                    target_port = p.device
                    break
            
            if target_port:
                try:
                    with serial.Serial(target_port, 9600, timeout=0.1) as ser:
                        ser.dtr = False
                        time.sleep(0.1)
                        ser.dtr = True
                        self.status_label.config(text=f"연결됨: {target_port}", fg="#4caf50")
                        while self.running:
                            if ser.in_waiting > 0:
                                line = ser.readline().decode('utf-8', errors='ignore').strip()
                                if line in self.settings:
                                    cmd_raw = self.settings[line]
                                    if ',' in cmd_raw:
                                        cmd_keys = cmd_raw.split(',')
                                        pyautogui.hotkey(*cmd_keys)
                                    else:
                                        pyperclip.copy(cmd_raw)
                                        pyautogui.hotkey('ctrl', 'v')
                            time.sleep(0.01)
                except:
                    self.status_label.config(text="연결 끊김. 재시도 중...", fg="#f44336")
            else:
                self.status_label.config(text="장치를 찾을 수 없습니다.", fg="#f44336")
            time.sleep(1)

if __name__ == "__main__":
    root = tk.Tk()
    app = MacroApp(root)
    root.mainloop()
